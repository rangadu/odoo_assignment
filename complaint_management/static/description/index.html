Complaint Management

The Complaint Management module provide a website form to submit the complaint.

Features

Website Form

Customers can submit the complaint through website.

Email Notifications

Email notifications will send when submitting the complaints and closing the complaints

Work Order Report

System users can download complaint work order report(DIN5008)

  • Configurations

Step 1

Website: Configurations > Settings > Complaint Responsible User

Step 2

Website: Complaints > Complaint Type

Step 3

Setting: General Setting > Companies > Document Layout > DIN 5008
